# TicTacToe

This is a repository for a simple TicTacToe React app.

Make sure you have Node.js and npm installed

# Cloning the repo

Type in an CLI

```
git clone https://github.com/Artyom-G/TicTacToe.git
cd TicTacToe
npm install
```

# Running the App

Type in an CLI 

`npm start`
